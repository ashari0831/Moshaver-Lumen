''' https://moshaver.herokuapp.com/api/v1/ '''

## Installation
### step 1:
run (terminal): 
``` composer install ```
### step 2:
run (termianl):
``` php artisan jwt:secret ```
### step 3:
complete your .env file like .env.example file for sending email and connecting to pusher
